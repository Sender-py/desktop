# Sender.py desktop tool 💫
A open-source, french, simple and fully free HTTP client.
![Preview](/Assets/preview.png)

## Options
GET Requests<br>
POST Requests<br>
PUT Requests<br>
DELETE Requests
GET Requests (with bearer token)<br>
POST Requests (with bearer token)<br>
PUT Requests (with bearer token)<br>
DELETE Requests (with bearer token)<br>

## Installation


Go to https://github.com/Sender-py/desktop/actions/workflows/build.yml and click on the latest workflow, then scroll and download for you Operating System.
Or: 
```bash
git clone https://github.com/Sender-py/desktop.git
cd desktop && cd src
pip install -r requirements.txt
python sender.py
```
